Если (if) остаток от деления равен 1, то выводим среднюю букву (middle), а если нулю (четное), выводим две средние буквы (срез от middle-1 middle+1)

Задание 2
Первым делом мы определили списки boys = ['x'...]. Здесь мы создаём два списка: boys и girls.
Списки содержат имена в виде строк.
Эти списки будут использоваться для формирования пар.
Далее мы определили функцию с помощью слова def и назвали ее make_pairs. Задали (boys, girls) как параметры функции, которые принимают списки имен.
Затем проверили длину списков с помощью команды len. != обозначает "не равно". Команда "return" завершает выполнение функции, если условие истинно.
Сортируем списки: sorted - встроенная функция Phyton, которая сортирует списки по алфавиту. Результаты сортировки сохраняются в новые переменные: boys_sorted и girls_sorted
Теперь формируем пары. zip(boys_sorted, girls_sorted) — функция, которая объединяет два списка в пары. list(zip(...)) — преобразует результат работы zip в список кортежей (пар).
Выводим результат: сначала выводим заголовок 'Идеальные пары', затем создаем цикл, который перебирает все пары из списка pairs (с помощью команды for). Далее команда print(f"{boy} и {girl}") выводит каждую пару в формате: "Имя_юноши и Имя_девушки".
f"{boy} и {girl}" — f-строка, которая подставляет значения переменных boy и girl в строку.
